# Clash Like Browser Game

Projeto full-stack de estrategia multiplayer inspirado em Clash of Clans.

## Requisitos

- Node.js 18+ recomendado

## Instalar

```bash
npm install
```

## Rodar

```bash
npm start
```

Abra no navegador:

```text
http://localhost:3000
```

## Recursos incluidos

- Cadastro e login persistentes
- Base em grade com construcoes e upgrades
- Ouro e energia gerados ao longo do tempo
- Treino de tropas
- Ataque PvP simplificado
- Clans, chat em tempo real e ranking
- Leaderboards globais e de clans
